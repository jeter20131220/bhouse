「我買了預售屋，有很多客變的想法！建設公司通知我要客變了，但我不知道這樣做好不好？」


「客變」是許多人在成家過程中遇到的第一個考題，對新手買房的屋主來說，要在短時間決定影響未來生活機 能的客變項目，甚至在對格局、生活需求沒有想像時，決定要做哪些更動，是有難度的。


  • 小寶優居的客變懶人包將幫你解決以下客變問題：
  • 1. 客變是什麼？為什麼要客變？
  • 2. 客變可更變的內容有哪些？
  • 3. 客變到底是省錢還是花錢？客變怎麼省錢？會產生哪些費用？
  • 4. 常見的客變需求與建議？我想這樣客變好不好？
  • 5. 設計師協助客變的流程？

客變是什麼？為什麼要客變？

「客變」是許多買預售屋的屋主期待的成家環節，在建設公司公告的客變期限內，屋主可以依照需求變更房屋 設計，就是所謂的「客戶變更設計」簡稱客變。 預售屋客變的最大優點就是可以讓房屋的格局與規劃更符合購屋者的需求，像是如果不想用建設公司提供的 廚具、不喜歡地板的款式，甚至有一道隔間牆不想施作或者移位，可以在客變階段提出變更，讓建設公司在施 作時依照你的規劃調整，降低之後拆掉重做的成本。 一般來說建設公司會有「客變規範資料」，讓客戶可以在客變期限前，在不影響建築結構安全，及管線通暢的情 況下，可以進行格局、建材上「有限度的變更」，針對可更動項目，建設公司會提出相對的施作報價，也可能因為 部分品項取消，可以退回設備與建材的項目金額。 客變是建設公司提供給客戶的服務之一，每個建案、建設公司可接受的客變都程度不同，並沒有規定建設公司 一定要讓客戶客變，所以小寶優居都會先建議想要客變的屋主「向建設公司先行詢問可客變範圍」。

客變可變更的內容有哪些？

許多成家新手誤會客變可以隨心所欲地調整內容，但事實上房屋的共同外觀以及每一戶共用的管路位置是無 法變更的，也就是窗戶、大門、陽台、樑柱、承重牆、廚房跟廁所位置、管道間都是無法變更的項目。

  • 常見可以進行客變的地方有以下：
  • 1. 隔間：屋內的格局，像是不做隔間牆、改門的位置、增加隔間…。
  • 2. 建材：地板（石英磚、木地板）、廚房衛浴磁磚、油漆…，通常是退料不退工。
  • 3. 設備：建設公司提供的廚具與衛浴設備，如果不喜歡可以退訂。
  • 4. 水電：插座、網路線、電話線、水龍頭與排水孔的位置…。

客變到底是省錢還是花錢？會產生費用？

常常可以聽見別人說客變可以節省裝修費用，事實上，客變可以幫你省下的是更動項目的拆除費用以及重複 支付的建材、設備費用，舉例來說，你想要把三房改成兩房，就可以在客變階段請建設公司少做一道牆，不僅省 下原先改兩房的拆除、清運費用，裝修時也可以更省時省力；但在客變階段，建設公司協助你修改與新增的項 目還是會有相應的費用產生，建設公司會就客變項目逐條列出費用與明細給屋主確認。

  • 一般來說，客變分成「退訂」、「修改新增」兩個方向：
  • • 退訂

    是針對原先施作的項目與設備做取消，像是不要鋪石英磚、不要門片、不要瓦斯爐、不要浴缸…。 這類的退訂，通常可以退回該項目建材與設備費用（退料不退工），但因為建設公司所附的建材與設備大多是 大量採購，往往價格是低於市價，退回的費用並不是依照市價計算之外，重新施作對屋主也會是一筆額外的支 出，往往會有「能退回的少，重做花更多」的現象，因此如果沒有特別的堅持，小寶優居一般會建議屋主保留建 設公司提供的項目。

  • • 修改新增

    指的是針對建設公司原先的規劃進行修改，像是增加或修改插座位置、調整隔間位置…這類的 修改可能伴隨的是費用的增加，決定客變時也需要透過圖面與建設公司溝通，避免客變成果與期待有落差。

常見的客變需求與建議

  • • 改格局、調整隔間

    拆除次臥房的隔間、移動隔間牆的位置改變房間大小…這類的更動就屬於更改格局，格局的更變建議以減法 為主，如果是增加隔間，小寶優居會協助屋主評估常見的家具尺寸後再進行格局更變，避免未來格局不合用、 甚至後悔的狀況。

  • • 改格局、調整隔間

    插座的位移、新增其實不一定要在客變時進行，根據小寶優居過往的實際經驗，完工後再調整插座位置，在費 用上並沒有太大差距，而且完工後依照櫃體、家具的尺寸進行規劃，也會比較精準、實用，所以插座的規劃不用 急於在客變階段完成。
    但若是有特殊的用電需求，像是用電量超出一般家庭，或者使用專業音響、安裝全熱交換器…需要獨立的迴路 就需要在客變階段提出討論。
